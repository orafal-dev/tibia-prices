# Tibia Prices

Tibia TC to PLN price converter for the character trade market — a browser extension that shows real-money (PLN) values next to Tibia Coins on the official character trade site.

## Install

- **Chrome**: `https://chromewebstore.google.com/detail/mcgfigcjalikhmdnphjbpnbnmcaolmam?utm_source=item-share-cb`
- **Firefox**: `https://addons.mozilla.org/en-US/firefox/addon/tibia-prices/`
